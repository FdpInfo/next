Frequently asked questions

  • Is fdp safe?
    we are open source so you can modify the code and compile yourself, we are trusted by the Community.
  • What does fdp mean?
    fdp means "Filho da puta" that is Portuguese.
  • What websites do we own?
    currently we only own fdpinfo.github.io, any other sites may have malware or be dangerous.
  • Will fdp work on my computer?
    fdp supports 64bit linux, mac and windows.
  • Is fdp liquidbounce
    fdp is a fork of liquidbounce so you get the benefits of liquidbounce while still getting updates and support.
  • Where is the fdp source code?
    fdp's source code can be found on github.

How to download!

These steps require that you have java edition and you have installed the base game, forge, and java.

1

Download

download the fdp jar file
2

Install

Move the jar file to your mod folder, you can use the official launchers directory button (folder icon) to go to the mod folder (under versions)
3

Running

Once fdp is in your 1.8 forge mod folder you can start the client.
